# Wafflepie's dotfiles

Run `dotfiles.sh` and stuff may happen to your computer.

## Quick install

```sh
git clone https://github.com/wafflepie/dotfiles.git ~/.dotfiles
~/.dotfiles/init.sh
```

Once installed, reinitialize by running `dot`.
